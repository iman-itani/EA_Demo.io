use case name
validate staffing request
primary business actor
contract manager
other participating actor
database system,placement department
other intrested stakeholder
description
this use case describe the validation of the staff request by the contract manger according to the information in the database system. If it's valid, the request is sent to the placement department. If it's not valid, the SRA client is notified.
precondition
The contract should be previously negotiated with SRA
trigger
A staffing request is received by SRA’s contract manager
business rules
The staffing request is valid if the contract has not expired, the type of professional or scientific employee requested is listed on the original contract, and the requested fee falls within the negotiated fee range.
